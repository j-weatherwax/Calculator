# Calculator

Basic Calculator based on the Windows 10 Calculator

## Installing
Download and install python [here](https://www.python.org/downloads/)

## Setup

1. Clone the repo
```sh
git clone https://github.com/j-weatherwax/Calculator.git
cd Calculator
```
2. Run the project
```python
python calc.py
```
